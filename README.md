Este projeto foi desenvolvido a partir de uma apresentação realizada em sala de aula com o objetivo de nos apresentarmos para a turma de forma dinâmica e interativa. No início da apresentação, mostramos algumas informações básicas sobre os integrantes do grupo, como nome, idade e cidade de origem, para que todos pudessem nos conhecer melhor. Entre os participantes estavam Ellen Rodrigues, de 18 anos, de São José dos Campos; Marina Yara, também com 18 anos, de São José dos Campos; Raíssa Esteves, de 20 anos, de Caçapava; e Juan Pablo, de 18 anos, também de Caçapava.

https://github.com/juanpablofreitas1809-boop/fonte-de-informa-o-e-banco-de-dados/blob/main/Captura%20de%20tela%202026-03-13%20204149.png

Após essa introdução, realizamos uma dinâmica chamada “1 verdade e 2 mentiras”, na qual uma pessoa apresenta três afirmações sobre si mesma, sendo uma verdadeira e duas falsas, enquanto os colegas tentam adivinhar qual delas é a verdade. As frases apresentadas foram: “Eu já trai duas vezes no mesmo ano”, “Eu já quebrei os dois braços” e “Eu recusei uma oferta para jogar no Corinthians”. Ao final da dinâmica, foi revelado que a afirmação verdadeira era “Eu já quebrei os dois braços”. A atividade teve como objetivo tornar a apresentação mais divertida, promover a interação entre a turma e criar um ambiente mais descontraído para que todos pudessem se conhecer melhor.
